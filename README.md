# Loggia Crew - VFX Studio Website

A simple, clean website for a visual effects studio built with HTML and Tailwind CSS. This project showcases the services, work, and process of Loggia Crew, a full-service VFX studio handling everything from on-set supervision to final delivery.

## Overview

This is a single-page website designed with a beginner-friendly approach to Tailwind CSS. It features a modern, responsive design with a color scheme of blue, jet black, white, and subtle orange accents.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Clean Navigation** - Easy-to-use navbar with utility bar for quick access
- **Portfolio Showcase** - Display work across Film, TV, Commercials, and Music Videos
- **Services Section** - Detailed breakdown of VFX capabilities
- **Process Timeline** - Step-by-step workflow from consultation to delivery
- **About Section** - Team story and studio philosophy
- **Insights Blog** - Case studies, tutorials, and news updates
- **Contact Form** - Simple contact form for project inquiries
- **Unsplash Integration** - High-quality images from Unsplash

## Getting Started

### Prerequisites

No build tools or dependencies required! Just a modern web browser.

### Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/loggia-crew-website.git
```

2. Navigate to the project directory:
```bash
cd loggia-crew-website
```

3. Open the HTML file in your browser:
```bash
open loggia-crew-website.html
```

Or simply drag and drop the `loggia-crew-website.html` file into your browser.

## Customization

### Changing Colors

The website uses Tailwind CSS utility classes. To change colors, modify the class names:

- Primary blue: `bg-blue-600`, `text-blue-600`, `border-blue-600`
- Orange accents: `bg-orange-500`, `text-orange-500`
- Black: `bg-black`, `text-black`

### Adding Images

Images are loaded from Unsplash. To use your own images:

1. Replace the Unsplash URLs with your image paths
2. Update the `src` attributes in the `<img>` tags

Example:
```html
<img src="your-image-path.jpg" alt="Description" class="w-full h-full object-cover">
```

### Modifying Content

All content is in plain HTML with the use of tailwind css for more customization.
